# 💳 Premium Subscription Tracker 🎬

A sleek, modern CRUD (Create, Read, Update, Delete) application to manage monthly subscriptions. Inspired by the hyper-polished, dark-theme UI/UX of modern fintech apps like CRED.

.The OTT Premium Subscription Tracker is a sleek, mobile-first web application designed to help users manage and monitor their recurring digital expenses (like Netflix, Spotify, and Amazon Prime). Built with Vanilla JavaScript and local storage, it features a hyper-polished, dark-mode user interface inspired by modern premium fintech applications, providing full CRUD functionality without the need for a complex backend.

   📝 Project Overview
     . With the rapid growth of Over-The-Top (OTT) media services and SaaS platforms, keeping track of monthly recurring expenses has become a common challenge. The OTT Premium Subscription Tracker is a client-side web application that solves this problem by allowing users to log, calculate, and manage their monthly digital subscriptions in one centralized dashboard.

.The project prioritizes a high-end, native-app feel, utilizing deep dark-mode aesthetics 🌙, custom typography 🔠, and smooth micro-interactions ✨ to deliver a premium user experience.


⚙️ Key Features & Functionality
.🔄 Complete CRUD Operations: Users can easily Create new subscription entries, Read their active list in a clean dashboard, Update pricing or service names, and Delete cancellations.

.💾 Persistent Data Storage: By leveraging the browser's native localStorage API, the application saves user data locally. This ensures that the subscription list remains intact even after the browser is closed or the page is refreshed, simulating a database experience without server-side latency.

.💎 Premium UI/UX Design: The interface abandons standard web forms in favor of a modern, fintech-inspired aesthetic. It features glass-like borders, high-contrast neon accents, and subtle CSS transform animations that provide tactile feedback when users interact with the cards and buttons.

.📱 Mobile-First Responsiveness: The layout is constrained and optimized for mobile viewports, mirroring the ergonomic feel of a native iOS or Android application.

.🛠️ Technical Implementation
.🏗️ Frontend Structure: HTML5 for semantic layout and accessibility.

.🎨 Styling: Advanced CSS3 utilizing CSS Variables (Custom Properties) for theme management, Flexbox for responsive alignment, and pseudo-classes for interactive state styling.

.⚡ Logic & State Management: Vanilla ES6+ JavaScript handles all DOM manipulation, form validation, and data structuring (JSON parsing/stringifying for local storage integration)
## ✨ Features

* **Full CRUD Functionality:** Easily add, view, edit, and remove your monthly subscriptions.
* **Persistent Storage:** Data is saved to your browser's `localStorage`, meaning you won't lose your list when you refresh the page.
* **Premium UI/UX:** Features a deep dark mode aesthetic, custom typography (Space Grotesk), glass-like borders, and smooth hover micro-interactions.
* **Mobile-First Design:** Optimized to look and feel like a native mobile application.

## 🛠️ Tech Stack

This project is built using strictly vanilla web technologies. No frameworks, no complicated build tools—just clean, readable code.

* **HTML5:** Semantic structure.
* **CSS3:** Custom variables, flexbox, and modern pseudo-class styling.
* **Vanilla JavaScript:** DOM manipulation and local storage management.
