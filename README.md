# 💳 Premium Subscription Tracker

A sleek, modern CRUD (Create, Read, Update, Delete) application to manage monthly subscriptions. Inspired by the hyper-polished, dark-theme UI/UX of modern fintech apps like CRED.

## ✨ Features

* **Full CRUD Functionality:** Easily add, view, edit, and remove your monthly subscriptions.
* **Persistent Storage:** Data is saved to your browser's `localStorage`, meaning you won't lose your list when you refresh the page.
* **Premium UI/UX:** Features a deep dark mode aesthetic, custom typography (Space Grotesk), glass-like borders, and smooth hover micro-interactions.
* **Mobile-First Design:** Optimized to look and feel like a native mobile application.

## 🛠️ Tech Stack

This project is built using strictly vanilla web technologies. No frameworks, no complicated build tools—just clean, readable code.

* **HTML5:** Semantic structure.
* **CSS3:** Custom variables, flexbox, and modern pseudo-class styling.
* **Vanilla JavaScript:** DOM manipulation and local storage management.
